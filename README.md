# Git基本ブランチモデル習得
Gitを実際の使用に近い形で使いこなせるように練習するためのレポジトリです。

最終的には、以下のブランチモデルを使用できるように毎日更新しております

- master
- develop/(バージョン番号）
- feature/（派生元バージョン番号）/（機能名）
- release/(バージョン番号）
- hotfix/（バグ識別名）
